# Parking Space Reservation System

A React frontend for registering users, browsing parking spaces, and managing reservations.

## Run locally

Install the dependencies and start the development server:

```bash
npm install
npm run dev
```

The frontend expects the reservation API at `http://localhost:8000/api` by default. Set `VITE_API_BASE_URL` when the API is hosted elsewhere.

## Available commands

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run lint` checks the source files with ESLint.
